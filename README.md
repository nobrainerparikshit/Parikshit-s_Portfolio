# AIYGO Website Clone

A static clone of [aiygo.org](https://aiygo.org) — the AI Youth Global Organization website.

Built with React, Vite, Tailwind CSS, and react-simple-maps.

## Pages

- **Home** — hero, interactive world map, origin story
- **Network** — member directory with search, partners
- **Resources** — 17 educational resources with tag filters
- **Projects**, **Blog**, **Sign Up**, **Information**, **Member Portal**

## Run locally

```bash
npm install
npm run dev
```

Open http://localhost:5173

## Build

```bash
npm run build
npm run preview
```

## Note

This is a visual/structural clone for development purposes. External links, forms, and resource URLs are placeholders unless pointed at the live site.
