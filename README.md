# Parikshit's Portfolio

A modern, responsive personal portfolio showcasing my work in artificial intelligence, robotics, research, software development, and engineering. The website serves as a central hub for my projects, achievements, publications, experiences, and ongoing initiatives.

Designed with performance, accessibility, and clean UI principles in mind, the portfolio is built using a modern React ecosystem and optimized for users across desktop and mobile devices.

## Features

* Modern and responsive interface
* Interactive user experience
* Fast client side navigation
* Clean and minimal design
* Dedicated pages for projects, research, and experiences
* Global network visualization
* Searchable member and resource sections
* Optimized performance using Vite
* Modular and reusable React components
* Mobile friendly layout

## Pages

### Home

### Projects

## Installation

Clone the repository.

```bash
git clone https://github.com/nobrainerparikshit/Parikshit-s_Portfolio.git
```

Navigate into the project directory.

```bash
cd Parikshit-s_Portfolio
```

Install dependencies.

```bash
npm install
```

Start the development server.

```bash
npm run dev
```

Open your browser and visit:

```
http://localhost:5173
```

## Production Build

Create an optimized production build.

```bash
npm run build
```

Preview the production build locally.

```bash
npm run preview
```
